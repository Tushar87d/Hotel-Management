🍽️ Hotel Menu Project

A simple Hotel Menu application created as a beginner project to display food items and their prices.
This project helps in understanding basic programming concepts and GitHub workflow.

* Features

Displays a hotel/restaurant menu

Shows food items with prices

Simple and easy-to-understand code

Beginner-friendly project

# Purpose

This project was built to:

Practice basic programming concepts

Learn how to create and manage a GitHub repository

Upload a first project on GitHub

~ Technologies Used

Language: Java

Platform:  Terminal

/ How to Run

Clone this repository

Open the project in your IDE or terminal

Run the program

View the menu displayed on the screen
